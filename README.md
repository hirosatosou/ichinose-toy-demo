# おもちゃのイチノセ サイトリニューアル（静的HTMLモック）

元サイト: https://www.toy-1.com/ （おもちゃのイチノセ／山梨県中央市西花輪）
作成日: 2026-06-15 ／ 種別: リニューアル提案用 静的モック（WordPress移植前提）

## 開き方
`index.html` をブラウザで開いてください。全ページ・全画像はこのフォルダ内で自己完結しています。

## ページ構成（10ページ → 5ページに統合・内容は全保持）
| 新ページ | 内容 | 元ページ |
|---|---|---|
| index.html | トップ（ロゴ/HAPPYコピー/サービス/カテゴリ/店長イチオシ/ブログ/アクセス） | index |
| products.html | 商品案内（男の子/女の子/ベビー知育/ホビーカード/テレビゲーム/バラエティの6カテゴリを1ページに統合、固定タブ付き） | 01boy〜06variety |
| guide.html | ご利用案内・店舗情報（営業時間/アクセス/地図＋ポイントカード＋法人購入を統合） | 08corporate + 09access + 07point |
| blog.html | ブログ（最新5記事カード→現アメブロ。WP移行で内部統合） | Amebaブログ |
| contact.html | お問い合わせ（フォーム＋電話/メール/地図） | メールのみ → フォーム化 |

「統合できるとこは統合」: 商品6ページ→1ページ（タブ）、会社情報＋アクセス＋ポイント→ご利用案内1ページ。
「内容は減らさない」: 各カテゴリの全コーナー説明、ポイント規約、法人購入の対象例、デュエルスペース利用時間まで保持。

## 素材
- ロゴ: `assets/ameblo_profile_logo.jpg`（アメブロのプロフィール画像＝当店の正規ロゴをそのまま使用）
- 商品/店内/店舗外観写真・店舗周辺マップ: `assets/`（元サイトから取得、151枚）
- ブログ記事サムネ: `assets/blog/`
- 元HTML/抽出テキスト: `source/` / `content/`（再構築の元データ、本番には不要）

## デザイン方針
- ブランド色はロゴ由来（レッド#ee3124／イエロー#ffc424／ターコイズ#16b6cf／グリーン／ブルー／クリーム地）
- 丸ゴシック（Zen Maru Gothic）＋Noto Sans JP。明るく楽しい玩具店らしい配色・角丸・装飾
- レスポンシブ（スマホ対応・ハンバーガーメニュー）、常時SSL前提

## WordPress 移植メモ（このモックでOKが出たら）
- テーマ化: header/footer を `header.php`/`footer.php`、共通CSSは `style.css` を流用
- 固定ページ: トップ（front-page.php）／商品案内／ご利用案内／お問い合わせ
- ブログ: WordPress標準の投稿（posts）へ。アメブロ記事は移行 or 当面はリンク
- お問い合わせ: Contact Form 7 等でフォーム稼働（現状デモは送信しません）
- 地図: Googleマップ埋め込み（住所: 山梨県中央市西花輪3508）
